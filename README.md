

# 📊 GlobalTech Solutions – Customer Management System

## 📌 Overview

This project is a **Python-based Customer Management System** developed as part of a Module 8 assignment. It demonstrates the use of:

* Dictionaries for data storage
* Data lookup techniques
* Aggregation and analysis
* Dictionary comprehensions
* Functions for validation and recommendations

The system simulates how a company like *GlobalTech Solutions* might manage customers, services, and project data.

---

## ⚙️ Features

### 🧾 Customer Management

* Store customer details (company name, contact, email, phone)
* Organize customers using unique IDs
* Update and validate customer information

### 💼 Service Management

* Maintain service categories with hourly rates
* Categorize services into pricing tiers (Basic, Standard, Premium)
* Apply rate adjustments (e.g., 10% increase)

### 📁 Project Tracking

* Assign projects to customers
* Track:

  * Project name
  * Service type
  * Hours worked
  * Budget
  * Status (active, completed, pending)

### 🔍 Data Lookup

* Retrieve specific customer information
* Handle missing data safely using `.get()`

### 📊 Analytics & Reporting

* Project cost calculations based on hourly rates
* Service usage analysis
* Financial summaries:

  * Total hours
  * Total budget
  * Average budget
  * Min/Max project budgets

### 📈 Aggregation & Insights

* Customer summary reports
* Budget analysis per customer
* Active customer filtering
* Project status tracking

### 🤖 Smart Features

* Customer validation function
* Budget analysis function
* Service recommendation system based on past usage
